**lint 178. 验证一个图是否是一个树**

给出 n 个节点，标号分别从 0 到 n - 1 并且给出一个 无向 边的列表 \(给出每条边的两个顶点\), 写一个函数去判断这张无向图是否是一棵树

**解决：**

* **bfs搜索：**从根节点出发向儿子节点进行广度优先搜索bfs，如果能遍历完所有的点，说明联通
* 如果能保证联通，就可以**通过判断边数是否为\(n - 1\)条来判断是否无环**

**代码思路**

* 首先判断边数是否为\(n - 1\)条，若不是则返回false
* 然后从根节点开始进行bfs搜索
